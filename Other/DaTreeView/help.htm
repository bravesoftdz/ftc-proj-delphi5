TDATREEVIEW

This component extends the ordinary treeview with some properties, methods and events to encapsulate cut & paste, drag & drop.

TYPES

TOnDragNodeEvent = procedure(Sender : TObject; Source, Destination : TTreeNode) of object;

TDATREEVIEW

Additional Properties

  • DragDropEnabled : boolean
    When this is true, drag & drop will happen automatically
  • CustomDragDrop : Boolean
    Whether the program will handle OnDragNode events

Additional Methods

  • procedure CutNode(Node : TTreeNode);
  • procedure CutNodes(Node : TTreeNode);
  • procedure DeleteNode(Node : TTreeNode);
  • procedure PasteNodeAt(Node : TTreeNode);
    Adds the formerly cut node(s) at the Node TTreeNode

Additional Events

  • property OnDeleteNode : TTVChangedEvent;
    Should handle the Data property for the node which is being deleted
  • property OnDragNodeEvent : TOnDragNodeEvent read FOnDragNodeEvent write FOnDragNodeEvent;
    Fired whenever drag & drop happens inside in the treeview if CustomDragDrop is true
