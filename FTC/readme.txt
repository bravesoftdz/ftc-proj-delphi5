
FOLDER TREE CREATOR 2.0

You can organize your boomarks and publish it on your web site using this tool.
The orginal Folder Tree was written using JavaScript & DHTML  from Marcelino Martins. 
This tool helps you builds this script and gives some additional features.

FILES

Included are the files

FTC2.EXE
FTC.CHM

FCTIMAGES/*.*

When deploying the files in the FTCIMAGES directory are copied into the 
deployment directory. Four files are written (by default they are called
tree.js, home.html, leftframe.html and index.html) 

REGISTRY

The program uses the entries in the registry

HKEY_LOCAL_MACHINE\SOFTWARE\Click It\Folder Tree Creator
